Acerca del Proyecto

Clasificador de Gomitas

Proyecto para la clasificación automática de gomitas y dulces similares mediante técnicas de aprendizaje profundo y redes neuronales convolucionales.

Tecnologías utilizadas

  • Python como lenguaje principal
  • TensorFlow para el entrenamiento del modelo
  • Keras para la construcción de la red neuronal
  • Flask para la API y la interfaz web
  • Bootstrap para el diseño responsive

Características principales

  • Clasificación de múltiples tipos de gomitas
  • API REST para integración con aplicaciones móviles
  • Interfaz web amigable
  • Soporte para añadir nuevas clases
  • Validación cruzada para mayor precisión

Estructura del proyecto

Archivo/Directorio Descripción
main.py Script principal para entrenamiento del modelo
predict.py Script para hacer predicciones individuales
add_class.py Herramienta para añadir nuevas clases al modelo
controller.py API REST y servidor web
app.py Punto de entrada para la aplicación
data/ Directorio con imágenes para entrenamiento y pruebas
models/ Modelos entrenados y archivos de metadatos
utils/ Funciones de utilidad y herramientas
